**Venv activation:**
`. venv/bin/activate`

**Deployment:**
Reference: https://fly.io

`fly deploy` (to deploy new changes to fly.io)

`fly launch` (for recreating Dockerfile and fly.toml)

`python3`

`reload(path_to_file)`
